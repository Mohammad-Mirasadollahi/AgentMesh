## About the project (historical)

AgentCore is a vendor-neutral **control plane** for registering, coordinating, governing, and observing external agent runtimes. Connected workers perform execution; the control plane owns tickets, routing, policy, approval, and audit.

For current docs and code, open:

**https://github.com/Mohammad-Mirasadollahi/AgentCore**
